RTG-code

Een gesloten, levende code

Onze code is geen gewone QR. Alleen de RTG-app kan hem maken en lezen, en hij vernieuwt zichzelf. Een foto is binnen een halve minuut waardeloos.

Scannen

De app leest de code en verifieert hem bij RTG. Een verlopen of vreemde code wordt geweigerd.

Waarom zo

  • Alleen onze app. De code is ondertekend met een sleutel die alleen op de RTG-server staat; een gewone QR-lezer ziet enkel "RTG1..." en kan er niets mee.
  • Dynamisch. Elke code vervalt na tientallen seconden en ververst vanzelf, dus een schermafdruk veroudert meteen.
  • In stijl. Bordeaux van het RTG-logo, met de lippen of het horloge in het hart.
