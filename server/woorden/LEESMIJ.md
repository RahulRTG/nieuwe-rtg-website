# Woordenlijsten voor het Woordduel

Gefilterd op speelbare woorden (2 t/m 15 letters, a-z, accenten
genormaliseerd), gededupliceerd en gecomprimeerd. De server leest ze bij
het eerste Woordduel-potje in en keurt elk gelegd woord.

- `nl.txt.gz` is Nederlands, afgeleid van de OpenTaal-woordenlijst
  (https://github.com/OpenTaal/opentaal-wordlist), licentie BSD 3-clause /
  CC-BY 3.0. Met dank aan Stichting OpenTaal.
- `en.txt.gz` is Engels, afgeleid van dwyl/english-words
  (https://github.com/dwyl/english-words, words_alpha), publiek domein
  (Unlicense).
